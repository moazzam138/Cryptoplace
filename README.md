# 🚀 Cryptoplace – Cryptocurrency Dashboard & Information Hub

## Overview

Cryptoplace is a modern web application built using React and Vite that provides users with a complete cryptocurrency information platform. The application allows users to track live cryptocurrency prices, analyze historical trends, read crypto-related news, explore educational articles, and manage basic user authentication.

The goal of this project is to combine market tracking, learning resources, and user interaction into a single platform for cryptocurrency enthusiasts, beginners, and traders.

---

## ✨ Features

## 📈 Real-Time Cryptocurrency Dashboard

The homepage displays the top cryptocurrencies with live market data fetched from the CoinGecko API.

Users can view:

- Cryptocurrency name
- Market rank
- Current price
- 24-hour percentage change
- Market capitalization

The dashboard also includes a search feature to quickly find specific cryptocurrencies.

---

## 💱 Multi-Currency Support

Users can switch between multiple currencies:

- USD
- EUR
- INR

Prices update dynamically based on the selected currency.

---

## 📊 Coin Details & Historical Charts

Each cryptocurrency has a dedicated details page showing:

- Live price
- Market ranking
- Historical price chart
- 10-day price trend analysis

Charts are implemented using React Google Charts.

---

## 📰 Crypto News Section

The project includes a crypto news section that provides curated articles and announcements from trusted sources such as:

- CoinDesk
- Cointelegraph
- CoinMarketCap

Users can open articles in a new tab to read full stories.

---

## 📚 Educational Articles

Cryptoplace includes educational content to help users understand blockchain and cryptocurrency topics.

Example articles include:

- What is Cryptocurrency?
- How Bitcoin Works
- Ethereum Explained
- Crypto Investment Basics

This makes the platform useful for both learning and market tracking.

---

## 👤 Authentication System

The project includes a basic client-side authentication system.

Features include:

- User Signup
- User Login
- Email/password validation
- Duplicate account prevention
- User data persistence using localStorage
- Automatic redirect after login

---
